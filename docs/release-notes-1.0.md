# Privora Release Notes — Version 1.0 (Public Launch)

We are thrilled to announce the official v1.0 public launch of **Privora**, your automated digital privacy hub. In this release, we deliver the complete core platform allowing individuals to discover, request removal of, and continuously monitor exposed personal files held by data broker systems.

## Core Features Launched

### 1. Centralized Privacy Dashboard
- Unified view of exposure metrics (Brokers Scanned, Active Exposures, Removals Processing, and Removals Completed).
- Circular Privacy Score Indicator (0-100%) showing a weighted index of PII leaks severity.
- Live activity timeline tracking background actions and database status synchronization.

### 2. Deep Scanning Engine
- Searches 80+ major public records lists, directories, and data broker registries (including Whitepages, Spokeo, BeenVerified, Radaris, and Intelius).
- Matches search strings for names, email addresses, phone records, and location history.
- Parameters hashing (SHA-256) to ensure search criteria are processed in-memory and never written to server log directories.

### 3. Automated Deletion (Opt-out Center)
- Pro-tier one-click automation to submit formal legal opt-out declarations to compliance systems.
- Free-tier step-by-step instruction sheets to request manual deletions.
- Interactive status updates (Pending -> Processing -> Completed or Refused).

### 4. Privacy Reports Archive
- Direct downloads of monthly compliance summary sheets as professional PDFs.
- Graphical history tracking score recovery trajectories.

### 5. Settings Management
- Autopilot toggling to run sweeps on a set frequency automatically.
- Stripe client subscription checkout interface.
- Custom notification configuration.
